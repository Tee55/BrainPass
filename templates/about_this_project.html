Brainwave authentication
using visual stimulus

By Mahidol wittayanusorn school

  Biometric authentication methods that have been used nowadays still have numerous potential security issues. Therefore, we investigate an authentication method using brainwave instead since brainwave is unique for each individual and it is difficult to be mimicked. In addition, we invent a software that uses brainwave as authentication method. Brainwave is applied to an innovation in various fields. The brainwave signals we observed is large-scale oscillations which can be measured by electroencephalography (EEG). In order to obtain brainwave signals, we use Visual Evoked Potentials (VEP) to stimulate the occipital lobe that controls visibility of human. People responds to the stimuli differently and has different brainwave signals, so it shows that the authentication method using brainwave can be used to authenticate identity of person. This authentication method has high level of security and it is difficult to be physically imitated. The software can also analyze the proper visual stimulator for each individual user. Software enhancement can significantly increase the accuracy when increasing the amount of data in database. In the experiment, we firstly collected EEG data which are stimulated by VEP using set of 260 Snodgrass-Vanderwart pictures and Checkerboard pattern. Afterward, we extract feature of EEG data and compute in user authentication using deep learning algorithm. In conclusion, the results show that our authentication method using brainwave demonstrates in achieving high verification accuracy and can be applied with some improvement in the future.
